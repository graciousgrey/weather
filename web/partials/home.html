Welcome to Symlex!

Symlex is a complete high-performance PHP framework based on Silex 2. Instead of Pimple, it uses the well known Symfony DI container. It promotes the strict use of dependency injection for implementing inversion of control and improved testability.

You can use Symlex with any JavaScript library and REST client or to output static HTML. The front-end example is based on AngularJS 1.6, Bootstrap, RequireJS and Bower for package management. A working example for command line applications is included as well.

After running bin/phing build, you can sign in as "admin@example.com" using the password "passwd" to access the user management.
